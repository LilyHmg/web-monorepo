# web 前端系统开发

- monorepo

## 开发环境

- Node.js Version `> 18.12`
- 包管理工具：`pnpm`

- 代码编辑器：`VSCode`

- 版本管理工具: `git`

### VSCode 配置参考

> 可以参考我的主题和插件

我将我的配置文件导出到了[.vscode/web.code-profile](./.vscode/web.code-profile)

点击 VSCode 左下角图标，找到导入配置文件，选择文件`web.code-profile`,创建配置文件，稍等一会儿安装好插件后会自动切换配置，也可手动切换。
